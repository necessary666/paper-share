# Awesome-GNN-Papers
本文主要是给入门GNN（图神经网络）的同学推荐一些经典论文。主要包含了同构图（Homogeneous Graph）和异构图（Heterogeneous Network）上的图神经网络算法，以及应用在图上的自监督、小样本学习、可解释性与应用等。

***持续更新中***
